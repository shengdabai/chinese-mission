## 已知 issue

### 1. DeepSeek 偶发 JSON parse 失败

复现率：**~5%**（m10 首轮失败 1/4 ≈ 25%，但全 20 个 mission 中只有 1 个挂掉 = 5%）

**根因**：DeepSeek 偶尔返回非严格 JSON（含尾随文字、unicode 引号等），`parseJSON()` 抛 SyntaxError → catch 走 500 fallback。

**影响**：用户看到一次 "Internal server error"，再发一次就好。

**建议修复**（不在本次 scope）：
- 给 `callAI` 加 1 次自动重试（temperature += 0.1 重试一次）
- 或者把 `response_format: { type: "json_object" }` 改为 prompt 内强制 + 抓 markdown fence regex

### 2. m19 slot 未识别

输入"我们约个时间开会吧" → slot=[]，但 NPC 反问主题。这是 prompt 设计选择 — m19 不强制立刻填 slot，先让用户提供更多上下文。**非 bug**。
